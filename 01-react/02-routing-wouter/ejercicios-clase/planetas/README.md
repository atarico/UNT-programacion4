# Ejercitación de routing paso a paso

## **Construir una app con 3 vistas**

- "/" -> Página principal
- "/listPlanetas" -> Listado de planetas
- "/planeta/:id" -> Detalle de un planeta

## Instalación

```bash
npm create vite@latest
cd mini-ejercicio-wouter
npm install
```

```bash
npm install wouter
```
